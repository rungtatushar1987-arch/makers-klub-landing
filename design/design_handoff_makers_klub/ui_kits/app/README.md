# Makers Klub — App UI Kit

Light cream / navy / ochre member-app direction. Five canonical screens.

## Screens
| File | What it is |
|---|---|
| `ScreenHome.jsx` | Klub home / monthly match feed |
| `ScreenBrief.jsx` | Brief onboarding (step 2/5) |
| `ScreenMatch.jsx` | Monthly match reveal |
| `ScreenSession.jsx` | Session detail w/ agenda timeline |
| `ScreenMemory.jsx` | **Past meetings + people** — stats, monthly groups, italic quote-cards, follow-up flags. This is how the platform helps members remember who they met. |

## System
- Surface: `--mk-cream`. Cards: white on `--border-1` hairlines.
- Action: `--mk-ochre` (yellow), used on CTAs + active states.
- Display headlines: Fraunces 800 with italic `<em>` accent in `--mk-lavender`.
- Bottom nav: navy pill, ochre active icon.
- Memory cards: italic Fraunces quote on cream-2 with ochre left-border.

Run `index.html` to see all five screens side-by-side as device frames.
